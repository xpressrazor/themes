1. File

Ctrl + x, Ctrl + f	Open file
Ctrl + x, Ctrl + s	Save

Ctrl + x, s    	 	Save All
Ctrl + x, k		Close Window/Buffer
Ctrl + x, Ctrl + c	Quit




2. Edit

C-a C-k C-k C-y C-y	Duplicate line
Ctrl + _    		Undo

Ctrl + g		Cancel Selection
Ctrl + w		Cut Selection
Ctrl + k		Cut Line
Ctrl + y		Paste
delete			Delete
Ctrl + d		Forward delete
Alt + delete		Delete word
Alt + d			Forward delete word
Ctrl + k		Delete line

Each tmux command is prefixed using the following key combination:

<Ctrl-b> or <C-b> (changed to C-a)

<C-b>%							: Split a window vertically 
<C-b>"							: Split the window horizontally 
<C-b>[Up, Down, Right, Left] 	: Move between active panes
<C-b>;							: Go to previous
<C-b><C-o>						: Rotate windows
<C-b><C-Up[Down][Left][Right]>  : Change pane sizes
<C-b><Meta-Up[Down][Left][Right]>
<C-b>x							: Closing a pane
<C-b>c							: Create new window
<C-b>n/p						: Switch to next window
<C-b>w							: Select window interactively
<C-b>&							: Close a window
<C-b>z							: Toggle zoom

Copy Mode

<C-b>[							: Scroll up the screen (use one of Up, Down, PgUp or PgDn to scroll up or down)
<C-b>]							: In scroll mode you can copy text from history and then paste it this key. To exit insert mode press ESC
<C-b>?							: Key bindings


Configuring Tmux (/etc/tmux.conf or ~/.tmux.conf)

Change the prefix key to <C-a>
set -g prefix C-a	# g = global option
unbind C-b
bind C-a send-prefix

<C-b>:							: Send commands


tmux new -s session_name		: Create a new tmux named session_name
tmux attach -t session_name		: Attaches to an existing tmux session
tmux switch -t session_name		: Switches to an existing session
tmux list-sessions				: tmux ls
<C-b>d							: detach window
<C-b> 0-9						: Switch to tmux window
<C-b> ,							: Rename window


tmux list-keys					: lists out every bound keys
tmux list-commands				: commands and arguments
tmux info						: lists every session, window, pane, pid
tmux source-file ~/.tmux.conf	: Reload current configuration
